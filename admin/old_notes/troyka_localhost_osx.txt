Troyka.com——————————Install Brew and Xcode command line developer tools:$ ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"Install a fresh Python, setup tools and pip$ brew install pythonInstall libraries for images processing etc$ brew install libtiff libjpeg libpngInstall virtual env$ pip install virtualenvClone the project from the git$ git clone git@git.assembla.com:troyka1.git troyka.comCreate a virtual env in project directory (python 2.6, because the server use this version)$ virtualenv --no-site-packages --python=python2.6 troyka.com/envActivate this$ source troyka.com/env/bin/activateGo to the project directory$ (venv)macbook:projects vit$ cd troyka.com/srcInstall all dependencies for troyka$ (venv)macbook:src vit$ pip install -r ../requirements.txt  Create a settings_local.py file$ (venv)macbook:src vit$ vim main/settings_local.pyDEBUG=TrueDATABASES = {    'default': {        'ENGINE':
